⚙️ ตั้งค่า Gold AI Live

🚀 คู่มือตั้งค่าเว็บไซต์

ทำครั้งเดียว ~10 นาที · เว็บของคุณอยู่ที่ github.io · เปิดจากมือถือได้ทุกที่

📋 สิ่งที่ต้องเตรียม
✕
บัญชี GitHub (ฟรี — สมัครที่นี่)
✕
Gemini API Key (ฟรี 1,500 ครั้ง/วัน — ขอได้ที่นี่)
○
Anthropic API Key (ไม่บังคับ ~$2-4/เดือน ทำให้ตัวค้านเก่งขึ้น)
○
MT5 เปิดค้างไว้ (ราคา realtime จากโบรก — ถ้าไม่มีใช้ Yahoo ได้)
1
สร้าง Repository บน GitHub
เข้า github.com/new
· ชื่อ: gold-ai-live
· เลือก Public (ต้องเป็น Public ถึงจะใช้ GitHub Pages ฟรีได้)
· กด Create repository
💡 เหตุผลที่ต้อง Public: GitHub Pages ฟรีใช้ได้กับ Public repo เท่านั้น ข้อมูลที่อัปโหลดขึ้นคือราคา+ข่าวสาธารณะ ไม่มีข้อมูลส่วนตัว
2
สร้าง Personal Access Token
ไปที่ Settings → Developer settings → Fine-grained tokens

กรอกข้อมูล:
· Token name: gold-ai-live
· Expiration: เลือก No expiration หรือ 1 ปี
· Repository access: Only select repositories → เลือก gold-ai-live
· Permissions → Contents → Read and write
· กด Generate token แล้วคัดลอก token ทันที!
⚠️ สำคัญ: Token จะแสดงแค่ครั้งเดียว ถ้าปิดหน้าไปแล้วต้องสร้างใหม่
3
ตั้งค่าไฟล์ .env
เปิดไฟล์ .env ในโฟลเดอร์ gold-ai-team แล้วใส่ค่า:
# .env — ห้ามแชร์ไฟล์นี้กับใคร!
GEMINI_API_KEY=AIzaSy...xxxxxx
GITHUB_TOKEN=github_pat_...xxxxxx
# ถ้ามี Claude key (ไม่บังคับ):
ANTHROPIC_API_KEY=sk-ant-...xxxxxx
# Telegram (ปิดอยู่ตอนนี้ เปิดได้ใน config.yaml)
TELEGRAM_BOT_TOKEN=
TELEGRAM_CHAT_ID=
4
ตั้งค่าไฟล์ config.yaml
เปิดไฟล์ config.yaml แก้บรรทัดนี้:
# เปลี่ยน USERNAME เป็นชื่อบัญชี GitHub ของคุณ
github_repo: "ชื่อบัญชีคุณ/gold-ai-live"
ตัวอย่าง: ถ้าบัญชี GitHub ของคุณชื่อ thana123 ให้ใส่ "thana123/gold-ai-live"
5
ติดตั้ง dependencies แล้วรันครั้งแรก
เปิด Command Prompt หรือ Terminal ในโฟลเดอร์นี้:
# ติดตั้ง (ทำครั้งเดียว)
pip install -r requirements.txt

# ทดสอบการเชื่อมต่อ
python main.py test

# วิเคราะห์ครั้งแรก + อัปโหลดขึ้นเว็บ
python main.py scan
ถ้าเห็น ✅ ทุกบรรทัด: ระบบพร้อมใช้งานแล้วค่ะ
6
เปิด GitHub Pages
1. เข้า repo ของคุณ: github.com/ชื่อบัญชี/gold-ai-live
2. คลิก Settings → Pages
3. Branch → เลือก main → Save
4. รอ ~2 นาที แล้วเปิด URL:
https://ชื่อบัญชีคุณ.github.io/gold-ai-live
💡 แชร์ลิงก์นี้ให้กลุ่มเทรดได้เลย — เปิดได้จากมือถือทุกที่ ไม่ต้องติดตั้งอะไร
7
รันอัตโนมัติ (แนะนำ)
ดับเบิลคลิกไฟล์ start_gold_bot.bat แล้วทิ้งหน้าต่างค้างไว้
ระบบจะส่งรายงานทุก 1 ชั่วโมง ตั้งแต่ 06:00–22:00
อยากให้เริ่มเองตอนเปิดเครื่อง: คลิกขวา start_gold_bot.bat → Create shortcut → กด Win+R พิมพ์ shell:startup → ย้าย shortcut ไปไว้ในโฟลเดอร์นั้น
